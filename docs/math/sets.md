# 集合
## 学习目标
1. 学会使用描述法来表示特征集合

## 符号
* **{}** 表示集合
* $a \in A$  a 元素属于 A
* $a \notin A$ a 元素不属于 A
* 集合的表示法
  1. **列举法** 采用 `A = {1,2,3}` 或 `B = {a1, a2, ..., an}` 列举集合所有元素
  2. **描述法**，描述集合中元素的特征用谓语公式描述，表示 $x \in A$ 当且仅当 P(x) 成立时
    * `A = {x | P(x) }`
    * `A = {x : P(x) }`
  3. **归纳法**
* $\empty$ 空集，$ \empty = \{  x| \not= x \} $

**注意采用描述法时，可结合逻辑公式和谓语公式说明集合特性**
